🚀 股癌分析系統轉移指南

當您更換新電腦時，請依照此簡報步驟重新建立您的 Workspace

1 下載專案程式碼

在新的電腦上，打開終端機 (Terminal) 或 PowerShell，將您的專案從 GitHub Clone 下來：

git clone https://github.com/fc861117-sketch/goodeyestocktracking.git
cd goodeyestocktracking

2 安裝系統必備工具

這個專案依賴強大的 AI 語音辨識模型，因此您必須在新電腦上安裝以下工具：

  • Python 3.10+: 請至 Python 官網下載並安裝 (安裝時記得勾選 Add Python to PATH)。
  • FFmpeg: 這是處理影片音軌的必備工具。
    • Windows 用戶可使用指令安裝：winget install ffmpeg
    • 安裝後請確保 FFmpeg 已加入系統的環境變數 (PATH) 中。
  • NVIDIA 顯示卡驅動與 CUDA (選用但強烈建議): 如果新電腦有 NVIDIA 顯示卡，請安裝最新版驅動程式以啟用 GPU 極速分析。

3 安裝 Python 套件

在專案資料夾內執行以下指令，安裝所有需要的程式套件：

pip install -r requirements.txt

*若您遇到 torchvision 找不到的問題，可以到 PyTorch 官網使用指令安裝 CUDA 版本的 torch 套件。

4 設定 API 金鑰 (非常重要)

⚠️ 為什麼要重新設定？
為了保護您的資安，config.json 密碼檔並沒有上傳到 GitHub！您必須在新的電腦上重新建立它。

1. 在專案資料夾內，找到 config.example.json。
2. 將它複製一份，並改名為 config.json。
3. 打開 config.json，填入您的 API Keys： 

{
    "gemini_api_key": "在這裡貼上您的 Gemini 金鑰",
    "github_token": "在這裡貼上您的 GitHub PAT (需有 repo 權限)",
    "github_repo": "fc861117-sketch/goodeyestocktracking"
}

5 享受全自動分析！

恭喜您！您的新環境已經架設完畢。現在您只需要執行以下指令，就能一鍵分析最新的影片並自動更新網頁：

python main.py analyze --limit 3

分析完成後，別忘了執行靜態網頁生成與推送到 GitHub：

python main.py generate-static
git add .
git commit -m "Auto update via new PC"
git push origin main
